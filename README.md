# ジョイゴル
## サイト概要
### サイトテーマ
プレーヤーが自分のスキルレベルに合ったコース選びができる、実践的なレビューサイト
​
### テーマを選んだ理由
私がこのテーマを選んだ理由は、自身の苦い経験に基づいています。  
私はゴルフが趣味で頻繁にコースをラウンドしていますが、ゴルフを始めたばかりの頃、コースの「プレー難易度」を把握しないまま難しいコースを予約してしまったことがあります。  
その結果、スコアが全くまとまらず、ゴルフに対するモチベーションが大きく低下してしまいました。  
こうした「スキルレベルとコース難易度のミスマッチ」は、私だけでなく、多くの初心者ゴルファーが経験する「壁」であり、ゴルフの本当の楽しさを知る前に競技から離れてしまう一因になっているのではないかと考えています。  
そこで、実際にラウンドしたプレーヤーが「プレー難易度」や「コースの戦略性」などを具体的にレビュー・共有できるサイトがあれば、プレーヤーが自身のスキルレベルに合ったコースを事前に選定する助けとなり、非常に役立つと考えました。  
このサイトが、これからゴルフを始める方々のコース選びをサポートし、ゴルフの楽しさを正しく体験してもらうことで、将来的にはゴルフ競技人口の維持・増加にも貢献できると信じ、このテーマを選定しました。
​
### ターゲットユーザ
- ゴルフを始めて1〜3年以内の人
- ゴルフスコアが安定せず100〜120前後でラウンドすることが多い人  
- これからゴルフを始めようとしている人  
- ハンディキャップを取得している競技志向の方々  
​
### 主な利用シーン
- 自分のレベルに合ったコースを探す時
- 自分の経験をレビューし、他の人を助ける時
- 自分のプレー結果を客観視する時
